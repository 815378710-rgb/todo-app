# 企业微信接入配置指南

## 一、获取 API 凭证（你需要做的）

### 步骤 1：获取 CorpID（企业 ID）

1. 打开 https://work.weixin.qq.com/wework_admin/loginpage_wx
2. 用管理员账号登录（就是你注册企业微信时的手机号）
3. 左侧菜单 → **我的企业** → 拉到最下面
4. 找到 **企业 ID**，复制保存

### 步骤 2：创建自建应用

1. 左侧菜单 → **应用管理** → **自建** → 点击 **创建应用**
2. 填写：
   - 应用名称：`待办提醒`（随便起）
   - 应用 Logo：随便传个图
   - 可见范围：选你自己
3. 创建完成后，进入应用详情
4. 记录：
   - **AgentId**（应用 ID）— 页面顶部能看到
   - **Secret** — 点击「查看」获取

### 步骤 3：设置回调 URL（等服务部署后再做）

1. 进入应用详情 → **接收消息** → 点击 **设置API接收**
2. 填写：
   - **URL**：`https://todo.maomaoxia.top/api/wecom/callback`
   - **Token**：随便填一个字符串，比如 `todo_remind_2026`
   - **EncodingAESKey**：点击「随机获取」
3. 点击「保存」— 这一步需要服务端已经部署并能响应验证请求
4. 记录 Token 和 EncodingAESKey

### 步骤 4：配置服务端

部署完成后，访问 `https://todo.maomaoxia.top`，在页面右下角点击 ⚙️ 设置按钮，填入：

| 字段 | 值 |
|------|-----|
| Corp ID | 步骤 1 获取的企业 ID |
| Agent ID | 步骤 2 获取的应用 ID |
| Secret | 步骤 2 获取的应用 Secret |
| Token | 步骤 3 设置的 Token |
| EncodingAES Key | 步骤 3 设置的 AES Key |
| 启用 | 打开 |

## 二、使用方式

配置完成后：

1. 在企业微信 App 里找到「待办提醒」应用
2. 发消息给它，比如：
   - `明天10点开会`
   - `3天后交报告`
   - `周五下午2点面试`
   - `买菜`
3. 机器人会回复确认，并在到时间推送提醒

## 三、常见问题

**Q：消息收不到？**
- 检查应用的「可见范围」是否包含你自己
- 检查回调 URL 是否配置正确（URL 必须是 HTTPS）
- 查看服务器日志是否有报错

**Q：回调验证失败？**
- 确保服务端已部署且能被外网访问
- 确保 Token 和 EncodingAESKey 与服务端一致
- Cloudflare Tunnel 需要正常运行

**Q：推送消息失败？**
- 检查 AgentId 和 Secret 是否正确
- 确认用户在应用的可见范围内
- 企业微信 API 有频率限制（每应用每分钟 200 条）
